# boba-editor

An advanced text editor based on [QuillJS](https://quilljs.com/), vaguely inspired by Tumblr's. Created for [BobaBoard](https://www.bobaboard.com).

[Demo on Netlify!](https://bobaeditor.netlify.app/)
